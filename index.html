Projects

Ongoing Research Projects (IIT Kharagpur)

NEUREQ: Neural Re-ranking Using Eligibility Question Modeling

A two-phase neural re-ranking framework for clinical trial retrieval that uses LLM-driven eligibility question answering to produce interpretable relevance signals. Structured answers to ten clinically motivated questions are used as features for a lightweight neural re-ranker, improving both effectiveness and transparency.

GitHub Repository

TCH_CLF: Reasoning-Augmented Teacher Classifier

A Longformer-based teacher model for clinical trial relevance modeling that integrates LLM-generated eligibility reasoning with structured trial attributes. The model is trained using a hybrid classification and ranking objective and serves as a strong neural baseline for TREC-style clinical trial re-ranking experiments.

GitHub Repository
